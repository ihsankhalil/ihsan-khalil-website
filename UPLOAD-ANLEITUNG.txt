UPLOAD-ANLEITUNG

Lade den kompletten Ordner "intern" in das öffentliche GitHub-Repository
"ihsan-khalil-website" hoch.

Danach lautet die Adresse:
https://ihsan-khalil.de/intern/

Enthalten:
- index.html
- styles.css
- app.js
- config.js
- .nojekyll
- robots.txt

Sicherheit:
- Der Publishable Key darf im Browser verwendet werden.
- Niemals Secret Key, Service Role Key oder GitHub Token in diese Dateien eintragen.
- Inhalte werden erst nach gültigem Supabase-Login geladen.
- Der Button zur automatischen Veröffentlichung funktioniert erst, nachdem die
  Supabase Edge Function "publish" eingerichtet wurde.
